Инструкция по запуску приложения GameProcess:

Перед тем как запустить консольное приложение GameProcess убедитесь, что у Вас установлено Java Runtime Environment(JRE), также прописаны Системные переменные в "Переменных средах".

Шаги для запуска:

	1) Поместить файл с расширением jar в нужную Вам папку.
	2) Открыть командую строку => В адресной строке "Проводника" прописать "cmd".
	3) Ввести java -jar GameProcess.jar
	4) Программа запускается в консоли
